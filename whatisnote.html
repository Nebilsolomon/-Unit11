what is Node.js and how is it related to JavaScript Node.js is a JavaScript runtime built on chrome v8 javascript engine Node.js is an open source server framework ,and it is free. it runs on various platforms (Windows, Linux, Unix, Mac OS X, etc.) and it uses javascript on the server. Node.js can generate dynamic page content ,so it can create, open, read, write, delete, and close files on the server Node.js can collect form. Moreover it can add, delete, modify data in your database. Node.js is built by google
